Install and activate venv
-------------------------------

### Create environment 
```bash
vierualenv -e python3 .env
```

### Activate
```bash
source .env/bin/activate
```

### Install dependencies
```bash
pip install -r requirements.txt
```

### run
```bash
python3 ./webserver.py
```
